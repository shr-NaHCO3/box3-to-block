# box3-to-block

## 什么是b2t?
    b2t来自于shyfcka大佬开发的box3-visual-programing(已被弃坑)。这个项目主要是基于bvp的进一步开发。

## b2t看起来是什么样子的？
    因为群主钱包余额为零，买不起服务器，所以b2t目前以浏览器插件的形式存在。
